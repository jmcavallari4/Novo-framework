# Founder OS — Sistema de Validación Paga™

> **Por Novolabs** | Versión 1.0

---

## Qué es esto

Founder OS es un asistente metodológico que guía a fundadores a través del **Sistema de Validación Paga™ (SVP™)**: un método de 7 etapas para validar ideas de negocio antes de construir nada.

Vive en tu computadora como una carpeta. No hay servidor, no hay cuenta, no hay suscripción.

---

## Cómo instalarlo

**Paso 1 — Descomprimir**
Descomprimí el ZIP en cualquier carpeta de tu computadora. Recomendamos `~/proyectos/founder-os/`.

**Paso 2 — Abrir en Claude Code**
Abrí Claude Code (la app de escritorio o la extensión de VS Code) y apuntá a la carpeta descomprimida.

**Paso 3 — Empezar**
Escribí en el chat:
```
/new-startup
```
El sistema te guía desde ahí.
